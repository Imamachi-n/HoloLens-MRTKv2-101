# HoloLens-MRTKv2-101
Sample projects for HoloLens1 by using MRTKv2

## Requirements
* Windows SDK 18362+
* Unity 2018.4.x (LTS) or Unity 2019.2.x (Latest)
* Visual Studio 2019

Unity and Visual Studio 2019 are installed by using Unity Hub.
